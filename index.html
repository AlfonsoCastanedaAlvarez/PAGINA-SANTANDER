App bancaria fullstack

Una banca móvil moderna, conectada a API y MongoDB.

Aplicación Android con registro, inicio de sesión, transferencias, pagos de servicios, perfil de usuario y movimientos en tiempo real.

Funcionalidades Tecnologías

Funcionalidades principales

La app permite administrar una cuenta bancaria simulada con datos reales almacenados en MongoDB Atlas mediante una API desplegada en Render.

Registro e inicio de sesión

Los usuarios pueden crear cuenta e ingresar con sus credenciales.

Saldo dinámico

El balance se actualiza automáticamente después de pagos y transferencias.

Transferencias

Permite enviar dinero a otros usuarios registrados dentro de la app.

Movimientos

Muestra historial de envíos, recibos y pagos de servicios.

Amenidades

Simula pagos de Spotify, Netflix, luz y agua.

Perfil

Consulta nombre, correo, teléfono y saldo actual del usuario.

Pantallas de la aplicación

Inicio

Pantalla principal con acceso a inicio de sesión y creación de cuenta.

Registro

Formulario para crear nuevos usuarios conectados a MongoDB.

Login

Validación de credenciales mediante API REST.

Dashboard

Muestra saludo, balance, acciones rápidas y navegación principal.

Últimos movimientos

Resumen de las operaciones más recientes del usuario.

Perfil

Datos del usuario y saldo consultados desde la API.

Movimientos

Historial completo de transferencias y pagos realizados.

Servicios y pagos

Pagos simulados de servicios que descuentan saldo real.

Transferencia

Envía dinero a usuarios registrados y actualiza ambos saldos.

Tecnologías utilizadas

Android Studio Java Retrofit Node.js Express MongoDB Atlas Render API REST
